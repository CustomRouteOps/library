# library
A collection of free React/TypeScript UI components organized by category with descriptions.

## Sources

- **21st.dev** — `web-components/react/` (various component categories)
- **Magic UI** (`magicui.design`) — `web-components/react/magicui/` — 71 animated/interactive React components across animations, backgrounds, buttons, cards, text-effects, and ui-elements. Install via `npx shadcn@latest add "https://magicui.design/r/<slug>"`
