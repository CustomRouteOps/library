# React UI Components

A curated collection of free React/TypeScript UI components sourced from [21st.dev](https://21st.dev).

## Folder Structure

```
react/
├── backgrounds/     # Background effects, gradients, patterns
├── buttons/         # Button variants and interactions
├── cards/           # Card layouts and displays
├── features/        # Feature sections and grids
├── footers/         # Footer layouts
├── heroes/          # Hero sections
├── inputs/          # Form inputs and AI chat inputs
├── navbars/         # Navigation bars and menus
├── pricing/         # Pricing sections and cards
├── testimonials/    # Testimonial sections
├── text-effects/    # Text animations and effects
└── accordions/      # Accordion components
```

## Tech Stack

All components use:
- **React** (with TypeScript)
- - **Tailwind CSS**
  - - **shadcn/ui** primitives
    - - **Framer Motion** (for animated components)
      - - **Lucide React** (for icons)
       
        - ## Source
       
        - Components sourced from [21st.dev](https://21st.dev) - a community-driven component library.
